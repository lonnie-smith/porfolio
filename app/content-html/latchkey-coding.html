This is a straightforward Angular 1.5 site with a couple of different routes. I try to follow John Papa’s magnificent Angular Style Guide pretty closely.

The data management techniques are interesting: I wrote an Angular service that pulls data from a Google spreadsheet, and another which queries the Google Books API to provide book data and book cover thumbnails.

This approach wouldn’t work for every project, but with a little tweaking (say, a Node script to pre-package the data in static JSON files), it could be a quick solution for a high-volume site with simple data needs.

The full code for this project is available on GitHub.
